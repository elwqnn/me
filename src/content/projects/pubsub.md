A fast, lightweight publish/subscribe message broker for Rust. Simple API, minimal setup, built on Tokio.  
Built to be used when services or tasks need to talk to each other without being tightly coupled, whether they live in the same process or across the network. Supports wildcards, queue groups for load balancing, and request/reply.
